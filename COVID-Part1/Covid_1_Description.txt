This COVID-Part1 is a subpart of the CovidPandemic_DataMining repository.
These data exploration, mining and visualization are done as per the blog created by Orange Data Mining.
(Link: https://orange.biolab.si/blog/2020/2020-04-02-covid-19-basic/)
It contains two main files:
i) The Covid_1 Orange Data Mining Workflow
ii) The Covid_1 resultant table created after COVID-19 Data Exploration.

Along with these, it also contains the line plot and scatter plot graph images as obtained during data exploration.

The dataset used here can be accessed from the link given below:
https://github.com/CSSEGISandData/COVID-19/blob/master/csse_covid_19_data/csse_covid_19_time_series/time_series_covid19_confirmed_global.csv

(Since it is a time-series-data, the data gets updated with time.)
